card-mod-helper

A Home Assistant tool for figuring out card-mod element selectors for themes
  • Open Chrome DevTools by right-clicking on a card and selecting Inspect
    or press ⌘⌥C (Mac) or Control+Shift+C (Windows, Linux)
  • Right-click on a DOM node in the Elements Panel and select Copy → Copy JS path
  • Paste in JS path and click "Convert JS path"
